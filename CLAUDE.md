# UltraContext Plugin

Context engineering for engineers who ship.

## Available Skills

- `/ultracontext` — Set up state-of-the-art context engineering for any repository. Interviews you about your workflow, analyzes your codebase, generates multi-level configs, hooks, session management, and token budgets.

## What It Does

UltraContext analyzes your repository and sets up the complete context engineering infrastructure:

1. **Interviews you** about tools, pain points, landmines, and workflow
2. **Analyzes your codebase** for non-obvious patterns and exact commands
3. **Scores your setup** with L0-L6 maturity and Hierarchy of Leverage frameworks
4. **Generates tailored configs** (AGENTS.md, CLAUDE.md, .cursor/rules, copilot-instructions.md)
5. **Sets up hooks** (auto-format, branch protection, PostCompact re-injection)
6. **Creates session management** (HANDOFF.md, .claudeignore, Document-and-Clear)
7. **Installs continuous sync** (pre-commit staleness detection)
8. **Reports token budgets** (know your context cost before conversations start)

## Core Laws

1. Less is more — every line must earn its place
2. Landmines, not maps — only what agents can't discover
3. Commands beat prose — snippets over paragraphs
4. Context is finite — 150-200 instruction budget
5. Progressive disclosure — layer context by need
6. Hooks for determinism — 100% compliance for critical rules
7. Negative instructions backfire — MUST Y, not don't X
8. Compact proactively — 60% safe, 85% hallucinating
